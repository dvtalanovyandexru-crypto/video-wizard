# 🎬 AI Video Director — Пошаговый мастер генерации видео

Интерактивный веб-интерфейс для создания видеоконтента с AI-озвучкой через API in One (Kling, Seedance, ElevenLabs) и n8n оркестрацию.

## 📋 Пошаговый процесс (9 шагов)

### Шаг 1: Сценарий и интерьер
- Загрузка файла сценария (TXT, DOC, PDF, MD)
- Или ручной ввод текста
- Подробное описание интерьера и локации

### Шаг 2: Диалоги персонажей
- Загрузка файла с диалогами
- Редактор диалогов с добавлением реплик
- Выбор количества персонажей (1-3)

### Шаг 3: Фото и видео персонажей
- Загрузка фото лица для каждого персонажа
- Загрузка видео-референса (опционально)
- Имя и описание персонажа

### Шаг 4: Уточнения по съёмке
- Описание одежды персонажей
- **Освещение** (с предустановкой — Естественное):
  - ☀️ Естественное ★
  - 🕯️ Тёплое
  - 💎 Холодное
  - 🎭 Драматичное
  - 🌫️ Мягкое
- **Цветовая гамма** (с предустановкой — Нейтральная):
  - 🟠 Тёплая
  - 🔵 Холодная
  - ⚪ Нейтральная ★
  - 🎬 Кинематографичная
- Позы, жесты, мимика
- Дополнительные указания

### Шаг 5: Выбор модели видеогенерации

**API in One (рекомендуется):**
- **Kling 3** ★ — высочайшее качество, плавность, физика (5-20 сек, 1080p)
- Seedance — быстрая генерация, хорошая детализация лиц
- Luma Dream Machine — кинематографичный стиль

**OpenRouter:**
- Claude Video — через Anthropic API
- GPT-4V Video — через OpenAI API

### Шаг 6: Выбор модели озвучки

**ElevenLabs:**
- **Multilingual v2** ★ — лучшее качество для русского языка
- Multilingual v1 — быстрее, чуть ниже качество
- Turbo v2.5 — максимальная скорость

**Настройки:**
- Скорость речи (0.5x — 1.5x, предустановка: 1.0x)
- Стабильность голоса (0 — 1, предустановка: 0.5)

### Шаг 7: Выбор голосов персонажей

**Мужские голоса:**
- Adam ★ (персонаж 1) — уверенный, глубокий
- Antoni (персонаж 2) — мягкий, дружелюбный
- Josh — энергичный, молодой
- Michael — авторитетный, зрелый

**Женские голоса:**
- Bella — тёплый, спокойный
- Rachel — профессиональный
- Elli — энергичный, молодой
- Matilda — драматичный, выразительный

*Кнопка "Прослушать" для предпрослушивания каждого голоса*

### Шаг 8: Параметры видео

**Длительность** (предустановка: 10 сек):
- 5 / **10** / 15 / 20 секунд

**Соотношение сторон** (предустановка: 16:9):
- **16:9** — горизонтальное
- 9:16 — вертикальное
- 1:1 — квадрат

**Количество сцен** (предустановка: 1):
- **1** — один непрерывный план
- 2 — смена ракурса
- 3 — монтажная нарезка

**Ракурсы** (множественный выбор, предустановка: Средний план):
- Крупный план
- **Средний план** ★
- Общий план
- Через плечо
- Двойной план

**Качество** (предустановка: Стандарт):
- Черновик — быстро, для теста
- **Стандарт** ★ — оптимальный баланс
- Высокое — максимальное качество

## 💾 Автосохранение

Черновик проекта автоматически сохраняется в `localStorage`. При следующем открытии интерфейса будет предложено загрузить сохранённый черновик (если прошло менее 24 часов).

## 🔗 Интеграция с n8n

### Пример webhook workflow:

1. **Webhook** — принимает POST запрос
2. **Validate** — проверяет API ключи
3. **Generate Video** — вызывает API in One (Kling/Seedance)
4. **Generate Audio** — вызывает ElevenLabs TTS для каждой реплики
5. **Sync** — синхронизирует аудио с видео
6. **Notify** — отправляет результаты обратно

### Ответ от n8n:

```json
{
  "status": "completed",
  "videoUrl": "https://cdn.example.com/video_123.mp4",
  "audioUrl": "https://cdn.example.com/audio_123.mp3",
  "projectUrl": "https://n8n.example.com/execution/123"
}
```

## 📱 Адаптивность

Интерфейс полностью адаптивен:
- Desktop (>900px) — оптимальная ширина
- Tablet (768-900px) — две колонки
- Mobile (<768px) — одна колонка, крупные элементы управления

## 🔐 Безопасность

- API ключи никогда не сохраняются в localStorage
- Ключи передаются только в POST запросе на webhook
- Все поля паролей скрыты (type="password")
